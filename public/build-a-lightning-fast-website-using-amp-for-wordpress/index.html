Build a Lightning Fast Website Using AMP for WordPress

29 October 2018

So you want a Super Fast Loading WordPress Website on Desktop and Mobile?

There are several approaches to creating a high-performance WordPress website. You can optimise an existing site for mobile speed, or preferably build a dedicated mobile site using AMP (Accelerated Mobile Pages) with mobile traffic redirected to it. AMP can also be used for both desktop and mobile rendering.

Solution: Install AMPFORWP Plugin (AMP for WordPress) to your WordPress Theme using AMP (Accelerated Mobile Pages)

Rather than building a custom WordPress theme, using AMP for WordPress allows you to create custom AMP layouts for mobile devices.

The Steps to Build an AMP WordPress Website

Step 1: Download AMP for WordPress Plugin

Download from ampforwp.com.

Step 2: Decide Which Elements You want to Use for Mobile

AMPFORWP offers additional paid extensions compatible with its framework, including plugins for contact forms (Contact Forms, Ninja Forms, Gravity Forms) and WooCommerce compatibility. The AMP Cache extension is particularly valuable, significantly improving mobile page load speeds.

Step 3: Which Theme Framework to Use?

AMPFORWP includes a theme framework for building AMP pages using built-in modules. You can alternatively use their pre-built themes or develop your own.

Step 4: AMP Settings

Configure your AMP settings by selecting which WordPress elements to enable (pages, posts, archives). Define SEO settings and install an AMP-compliant SEO plugin like Yoast or All in One SEO. Enable minification to further enhance performance. The latest version includes an option to disable AMP for iPad portrait mode if your theme's breakpoint is 1024px. Enable WooCommerce compatibility if applicable.

Step 5: Build your pages

Construct AMP pages according to your design specifications, using the Design global settings to add custom CSS.

Step 6: Check Your Pages are indexed in Google

After several days, verify that AMP pages are properly indexed in Google Search Console, identifying and resolving any AMP errors, warnings, or issues.
