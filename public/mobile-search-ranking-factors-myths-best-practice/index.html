Mobile Search Ranking Factors, Myths & Best Practices

13 February 2013

Even the most optimistic digital analysts recognise that mobile search traffic will dominate in coming years. Understanding key factors for mobile search success is essential.

Mobile Search Best Practices

1. Responsive Design or Not?

Responsive web design is the recommended approach, using CSS media queries to control HTML content across different devices. While Google supports separate mobile URLs (e.g. www.acme.com and m.acme.com), responsive design remains the preferred solution: "the general consensus in the mobile SEO industry is to deploy a responsive design as first choice solution."

A separate mobile site is acceptable but not necessary for ranking improvements.

2. Use of Robots.txt. NOT!

Avoid blocking mobile URLs with robots.txt. Instead, utilise canonical tags for duplicates and redirect smartphone URLs. Ensure mobile homepage content is unique and clearly labelled with "mobile" in the title tag and body text.

3. How Important Is HTML Code Compliance

Mobile search engines don't penalise invalid HTML more than valid code, so achieving perfect XHTML compliance isn't a priority.

4. Usability NOT Rankability

Navigation ease and fast loading matter for user experience but don't directly improve rankings.

5. Presence and Relevance of Content

Keywords matching mobile search queries help rankings. Optimise content appropriately for your industry.

6. Link Authority

Link building and page authority remain ranking factors across all website types, including mobile sites.

7. Focus On The User, NOT the device

The best mobile experiences prioritise ergonomics, usability, and solving actual user problems.

Mobile Search Ranking Factors

A. Most Important

  1. Website correctly redirects to Googlebot Mobile or Smartphone Googlebot
  2. Proximity of business location to searcher's device location
  3. Content relevance to searcher's query

B. Important

  1. Mobile sitemap exists
  2. Presence of mobile site (responsive or dedicated)

C. Of Limited Importance

  1. Use of "mobile" in page title tags
  2. Presence of mobile app in Google Play
