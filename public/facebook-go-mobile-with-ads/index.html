FaceBook Go Mobile With Zero

28 June 2010

Facebook recently introduced a lightweight mobile version called Zero, accessible via zero.facebook.com. This stripped-down variant eliminates bandwidth-intensive media images, presenting instead a simplified text-based interface.

Performance and Accessibility

Testing the service on an HTC mobile device over an O2 network demonstrated solid functionality, allowing swift access to inboxes and messaging features. The initiative represents a strategic move for Facebook to expand its social networking reach through mobile operators who can pre-install the application on new devices.

Market Implications

Mobile carriers and network providers can now capitalize on this offering by potentially charging users for the richer, multimedia-enabled version while providing the basic text iteration at no cost. With over 100 million Facebook users already accessing the platform via mobile devices, this represents a significant market opportunity.

Context Within Existing Services

Facebook Zero builds upon the previously established Facebook Lite service, which targeted users in developing nations and rural areas with limited broadband connectivity, particularly throughout the United Kingdom.
