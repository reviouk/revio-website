Calendar Plugin: a Professional, Well Thought Out Add On for WordPress

17 September 2014

After evaluating multiple WordPress calendar solutions, we selected 'WordPress Pro Event Calendar' from CodeCanyon as our recommended plugin.

Key Features

The plugin prioritises responsive design across all devices. It includes Google Maps integration and permits users to contribute their own events. The tool supports iCal compatibility, enabling import into Outlook and mobile calendar applications, plus MailChimp subscription integration.

The calendar offers dual colour schemes — light and dark versions — with customisable CSS styling options.

Setup Process

Configuration is straightforward:

  1. Create an 'Events Page' and assign a calendar to it.
  2. Access configurable settings through collapsible menus and checkboxes.
  3. Insert the shortcode (example: [dpProEventCalendar id=1]).
  4. Publish the page.

The implementation requires minimal technical effort while delivering professional calendar functionality for WordPress sites.
