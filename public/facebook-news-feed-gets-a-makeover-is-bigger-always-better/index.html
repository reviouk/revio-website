Facebook News Feed Gets a Makeover: is bigger always better?

12 March 2013

"Leave it alone Mr Zuckerberg. There's no need to change something that's working perfectly well!" The redesign focuses on restructuring how users view shared content and browse stories based on topics and liked items. Following user complaints about cluttered feeds, Facebook implemented this overhaul.

Design Approach

The platform drew comparisons to the Flipboard application, partitioning content through improved structured layout organised by topics and sources. To access the new interface, users could apply and wait in queue on Facebook's dedicated application page.

Current State vs. New Structure

Previously, news feeds blended images, videos, friend posts, and comments into one vertical stream. The updated version resembles iPad newspaper applications like The Times, featuring organised sections: front-page headlines, politics, sports, fashion, and celebrity content.

Key Interface Changes

  • Upgraded navigation panel with larger icons and text
  • Dedicated music feed
  • Photo/image feed
  • Message feed
  • Enhanced topic-based organisation

Strategic Implications

Monetisation Goals

The redesign aims to increase user engagement and platform monetisation through advertising. With users averaging roughly twenty minutes daily on Facebook, the goal involves extending session duration and interaction time.

Data Collection & Targeting

User customisation choices reveal preferences, enabling Facebook to create a more targeted advertising platform. This benefits businesses through improved ad visibility and stronger viewer connections, ultimately increasing click-through rates.

Three-Party Benefits

All stakeholders gain: users enjoy improved organisation, advertisers access better targeting, and Facebook strengthens share value and platform viability.
