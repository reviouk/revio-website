Link Building Myths and Secrets for Success

27 November 2009

Link building tips revealed

There are numerous online articles about link building strategies, though few specifically address creating natural-looking links with sustained first-page search results. Rankability has analysed link building patterns and ranking behaviours to provide guidance on effective practices.

Myth 1: keyword-only anchor text

Reality: Top-performing websites typically avoid overusing keyword anchor text. These sites maintain varied anchor text, with approximately 75% using domain names or company names instead.

Recommendation: Vary your anchor text. Build a foundation using your domain/company name before targeting keyword anchor text. This approach appears more natural and avoids establishing unsustainable linking expectations.

Myth 2: off-topic links

Reality: Google understands webmasters cannot control incoming links. Off-topic links from different niches are acceptable alongside on-topic links for robust search rankings.

Recommendation: Link to relevant websites within your niche; they may reciprocate if they find your content valuable.

Myth 3: no-follow links

Reality: No-follow links are essential for a natural link profile. Pursuing only do-follow links appears suspicious. Web 2.0 and social bookmarking sites with no-follow attributes create authentic linking patterns.

Myth 4: forum links

Reality: Forum links work well for low-competition keywords when combined with other sources. Niche forum links provide value and mature over time.

Recommendation: Select quality forum sites carefully. Review existing links on target pages and avoid excessive outbound links. Create new topics, provide value, solve problems, and vary anchor text.

Myth 5: rapid link building penalties

Building numerous quality links quickly (link velocity) typically generates only short-term ranking improvements and sets unsustainable expectations. Gradual, varied approaches prove more effective long-term.

Example: for targeting "scuba diving holidays" at www.scubydoobydoo.com, use varied anchor text including: the domain URL, brand name, domain with keywords, exact match phrases, and mixed formats.

Key strategy

Domain name anchor text should exceed targeted keyword anchor text for a natural appearance. Combined with solid on-page optimisation, this approach supports top-3 rankings. Many top-ranking domains employ this exact method.

Common problem scenario

Some SEO firms create short-term ranking improvements through aggressive link velocity, then demand additional payment for maintenance. This pattern indicates reliance on unsustainable tactics rather than diversified, multi-source strategies including social media integration.
